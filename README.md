# Utilities

Coleção de scripts, patches e ferramentas de apoio ao ambiente de desenvolvimento da [singularidade.digital](https://singularidade.digital).

## Conteúdo

| Diretório | Descrição |
|-----------|-----------|
| [`pgmodeler/`](pgmodeler/) | Build from source do pgModeler 1.2.1 com patch para PostgreSQL 18 |

## Sobre

Este repositório centraliza utilitários que não pertencem diretamente aos projetos de produto, mas são necessários para configurar e manter o ambiente de desenvolvimento da equipe.

Cada diretório é independente e contém seu próprio `README.md` com instruções de uso.

## Requisitos gerais

- **SO**: Linux Mint 22.1+ / Ubuntu 24.04+
- **Ferramentas**: `git`, `bash`

Requisitos específicos de cada utilitário estão documentados no respectivo `README.md`.
